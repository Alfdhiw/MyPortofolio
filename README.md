# My Portfolio

Ini adalah *source code* website portofolio pribadi saya yang murni menggunakan HTML statis, CSS, dan JavaScript murni. Desain antarmuka portofolio ini pada awalnya dikompilasi menggunakan Tailwind CSS.

## Cara Menjalankan

Karena project ini sekarang adalah website statis murni tanpa framework *backend* (tidak menggunakan Laravel atau PHP), Anda tidak perlu menggunakan web server, Docker, atau instalasi yang rumit. 

Anda cukup **klik dua kali pada file `index.html`** di komputer Anda, dan website portofolio ini akan langsung terbuka dan berjalan sempurna di browser kesayangan Anda (Google Chrome, Firefox, dll).

## Deployment (Hosting)
Anda juga bisa meletakkan (deploy) project ini secara langsung di layanan hosting statis gratis dengan sangat mudah karena tidak ada *backend* yang dibutuhkan. Contoh layanan yang bisa digunakan:
- GitHub Pages
- Netlify
- Vercel

Cukup *push* ke GitHub dan hubungkan ke layanan di atas, portofolio Anda siap untuk online!
